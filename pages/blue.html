addExtraIcon

addExtraIcon(tag:String, char:String, ?xOffset:Float = 0, ?yOffset:Float = 0, ?isPlayer:Bool = false):Void

Adds an extra icon to the healthbar, with x and y offsets, using isPlayer to flip.

  • tag - Sprite name.
  • char (Optional) - Icon Name, defaults to face.
  • xOffset (Optional) - X offset, used to position the icon
  • yOffset (Optional) - Y offset, used to position the icon
  • isPlayer (Optional) - Is the icon a player icon?

Examples:

  • addExtraIcon("grandma", "pico", 25, -25, true) Will add Pico's icon facing the left, 25 pixels right and 25 pixels up from the healthbars center.

makeLuaBar

makeLuaBar(tag:String, varToFollow:String, ?image:String = "timebar", ?x:Float = 0, ?y:Float = 0, ?leftImage:String, ?rightImage:String, ?direction:String = 'ltr'):Void

Adds a bar, using a variable set using setVar()

  • tag - Sprite name.
  • varToFollow - Variable to follow. Use setVar to set and update the bar.
  • image (Optional) - Bar image, used to determine the fill.
  • xOffset (Optional) - X, used to position the bar on the X axis.
  • yOffset (Optional) - Y, used to position the bar on the Y axis.
  • leftImage (Optional) - Sets an image used for the left bars fill. (useful for weirdly shaped bars)
  • rightImage (Optional) - Sets an image used for the right bars fill. (useful for weirdly shaped bars)
  • direction (Optional) - Sets how the bar will fill. The options are: 'ltr' (left to right), 'rtl' (right to left), 'dtu' (down to up), and 'utd' (up to down).

Examples:

  • makeLuaBar('attackBar', 'attackCharge', 'attackBarImage', 1250, 40, 'attackFill', 'attackFill', 'dtu') Will add a bar on the right side of the screen, using the attackCharge variable to determine its fill.

setBarColors

setBarColors(tag:String, ?leftColor:FlxColor = "FFFFFF", ?rightColor:FlxColor = '000000'):Void

Sets a lua bars colors.

  • tag - Sprite name.
  • leftColor (Optional) - The color of the left bar.
  • rightColor (Optional) - The color of the right bar.

Examples:

  • setBarColors('myBar', 'FA2A2A', '696969') Will set the bar "myBar" colors to red and grey.

setBarBounds

setBarBounds(tag:String, ?min:Float = 0, ?max:Float = 1):Void

Sets a sets a bars minimum and maximum value.

  • tag - Sprite name.
  • min (Optional) - The minimum value a bar can have, if your value is under or equal to this it makes the bar look empty.
  • max (Optional) - The maximum value a bar can have, if your value is over or equal to this it makes the bar look full.

Examples:

  • setBarBounds('myBar', 0, 5) Will set the bar "myBar" bounds to 0 and 5.

makeLuaCharacter

makeLuaCharacter(tag:String, ?json:String = 'bf', ?x:Float = 0, ?y:Float = 0, ?chart:String = '', ?skipDance:Bool = false, ?danceEveryNumBeats:Int):Void

Makes a new character, tied to a chart with an option to change the amount of beats before dancing.

  • tag - Sprite name.
    • "
  • json (Optional) - name of the character to load.
  • x (Optional) - X position.
  • y (Optional) - Y position.
  • chart (Optional) - Name of the chart file to load in data/songName/
  • skipDance (Optional) - Whether the character idles or not.
  • danceEveryNumBeats (Optional) - the number of beats before the character idles.

Examples:

  • makeLuaCharacter('picoSpeak', 'pico-speaker', 0, 0, 'picoSpeaker', true, 0) Will make a character using the 'pico-speaker' json following the chart 'picoSpeaker'.

setSpriteRGB

setSpriteRGB(tag:String, ?r:Dynamic = 'FF0000', ?g:Dynamic = '00FF00',?b:Dynamic = '0000FF',?a:Array = null):Void

Adds the RGB shader onto a lua sprite, can be used again to change color and alpha values.

  • tag - Sprite name.
  • r (Optional) - Int or String of a color, used to shade the R of a sprite.
  • g (Optional) - Int or String of a color, used to shade the G of a sprite.
  • b (Optional) - Int or String of a color, used to shade the B of a sprite.
  • a (Optional) - Sets the alpha of each color.

    • Examples:

    • setSpriteRGB('boyfriend', '696969', 'FA2A2A', '00FF00', {0.5, 0.2, 1}) Will set boyfriends rgb shader colors to grey, red, and green, while setting the transparency of each color.
