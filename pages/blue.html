addExtraIcon

addExtraIcon(tag:String, char:String, ?xOffset:Float = 0, ?yOffset:Float = 0, ?isPlayer:Bool = false):Void

Adds an extra icon to the healthbar, with x and y offsets, using isPlayer to flip.

  • tag - Sprite name.
  • char (Optional) - Icon Name, defaults to face.
  • xOffset (Optional) - X offset, used to position the icon
  • yOffset (Optional) - Y offset, used to position the icon
  • isPlayer (Optional) - Is the icon a player icon?

Examples:

  • addExtraIcon("grandma", "pico", 25, -25, true) Will add Pico's icon facing the left, 25 pixels right and 25 pixels up from the healthbars center.

makeLuaBar

makeLuaBar(tag:String, varToFollow:String, ?image:String = "timebar", ?x:Float = 0, ?y:Float = 0, ?leftImage:String, ?rightImage:String, ?direction:String = 'ltr'):Void

Adds a bar, using a variable set using setVar()

  • tag - Sprite name.
  • varToFollow - Variable to follow. Use setVar to set and update the bar.
  • image (Optional) - Bar image, used to determine the fill.
  • xOffset (Optional) - X, used to position the bar on the X axis.
  • yOffset (Optional) - Y, used to position the bar on the Y axis.
  • leftImage (Optional) - Sets an image used for the left bars fill. (useful for weirdly shaped bars)
  • rightImage (Optional) - Sets an image used for the right bars fill. (useful for weirdly shaped bars)
  • direction (Optional) - Sets how the bar will fill. The options are: 'ltr' (left to right), 'rtl' (right to left), 'dtu' (down to up), and 'utd' (up to down).

Examples:

  • makeLuaBar('attackBar', 'attackCharge', 'attackBarImage', 1250, 40, 'attackFill', 'attackFill', 'dtu') Will add a bar on the right side of the screen, using the attackCharge variable to determine its fill.
