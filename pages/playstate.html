PlayState Functions

General functions related to the Game State, Song and Cutscenes.

startCountdown

Usage: startCountdown()

Starts the countdown, should be used after finishing your Pre-Song Cutscene.

endSong

Usage: endSong()

Ends the song, can be cancelled through the Lua/HScript callback "onEndSong" to make cutscenes.

getSongPosition

Usage: getSongPosition()

Returns the current song position, in milliseconds.

restartSong

restartSong(?skipTransition:Bool = false):Void

Restarts the Song, pretty self explanatory.

  • skipTransition (Optional) - Skips fade in/out transition, defaults to false.

exitSong

exitSong(?skipTransition:Bool = false):Void

Quits the Song and goes back to Freeplay/Story Mode based on where you started.

  • skipTransition (Optional) - Skips fade in/out transition, defaults to false.

loadSong

loadSong(?name:String = null, ?difficultyNum:Int = -1):Void

Loads the chart for a song.

  • name (Optional) - Song Name (ex. "Bopeebo", "Lit Up").
  • difficultyNum (Optional) - Week's Difficulty Number (0 = Easy, 1 = Normal, 2 = Hard), use -1 to keep it the same as current.

triggerEvent

triggerEvent(name:String, ?value1:String = '', ?value2:String = ''):Bool

Calls an event and uses two arguments for it, exactly like the Chart Editor does.
Should always return "true".

  • name - Event Name.
  • value1 (Optional) - Does the same as what the Event says in the Chart Editor.
  • value2 (Optional) - Does the same as what the Event says in the Chart Editor.

Examples:

  • triggerEvent('Hey!', 'BF', 0.6) - Calls "Hey!" event.

addExtraIcon

addExtraIcon(tag:String, char:String, ?xOffset:Float = 0, ?yOffset:Float = 0, ?isPlayer:Bool = false):Void

Adds an extra icon to the healthbar, with x and y offsets, using isPlayer to flip.

  • tag - Sprite name.
  • char (Optional) - Icon Name, defaults to face.
  • xOffset (Optional) - X offset, used to position the icon
  • yOffset (Optional) - Y offset, used to position the icon
  • isPlayer (Optional) - Is the icon a player icon?

Examples:

  • addExtraIcon("grandma", "pico", 25, -25, true) Will add Pico's icon facing the left, 25 pixels right and 25 pixels up from the healthbars center.

makeLuaBar

makeLuaBar(tag:String, varToFollow:String, ?image:String = "timebar", ?x:Float = 0, ?y:Float = 0, ?leftImage:String, ?rightImage:String, ?direction:String = 'ltr'):Void

Adds a bar, using a variable set using setVar()

  • tag - Sprite name.
  • varToFollow - Variable to follow. Use setVar to set the variable.
  • image (Optional) - Bar image, used to determine the fill.
  • xOffset (Optional) - X, used to position the bar on the X axis.
  • yOffset (Optional) - Y, used to position the bar on the Y axis.
  • leftImage (Optional) - Sets an image used for the left bars fill. (useful for weirdly shaped bars)
  • rightImage (Optional) - Sets an image used for the right bars fill. (useful for weirdly shaped bars)
  • direction (Optional) - Sets how the bar will fill. The options are: 'ltr' (left to right), 'rtl' (right to left), 'dtu' (down to up), and 'utd' (up to down).

Examples:

  • makeLuaBar('attackBar', 'attackCharge', 'attackBarImage', 1250, 40, 'attackFill', 'attackFill', 'dtu') Will add a bar on the right side of the screen, using the attackCharge variable to determine its fill.

setBarColors

setBarColors(tag:String, ?leftColor:FlxColor = "FFFFFF", ?rightColor:FlxColor = '000000'):Void

Sets a lua bars colors.

  • tag - Sprite name.
  • leftColor (Optional) - The color of the left bar.
  • rightColor (Optional) - The color of the right bar.

Examples:

  • setBarColors('timeBar', 'FF0000', '00FF00') Will set the timeBar colors to red and green.
  • setBarColors('timeBar', 'red', 'lime') Will set the timeBar colors to red and green.

setBarBounds

setBarBounds(tag:String, ?min:Float = 0, ?max:Float = 1):Void

Sets a sets a bars minimum and maximum value.

  • tag - Sprite name.
  • min (Optional) - The minimum value a bar can have, if your value is under or equal to this it makes the bar look empty.
  • max (Optional) - The maximum value a bar can have, if your value is over or equal to this it makes the bar look full.

Examples:

  • setBarBounds('myBar', 0, 5) Will set the bar "myBar" bounds to 0 and 5.

startDialogue

startDialogue(dialogueFile:String, ?music:String):Void

Loads a Dialogue .JSON inside the Chart folder, handles translation loading automatically.

  • dialogueFile - Dialogue file .JSON name.
  • music (Optional) - Plays a background music inside "music/" folder.

Examples:

  • startDialogue('dialogue') - Loads "dialogue.json" inside the chart folder, will prefer "dialogue_pt-BR.json" if you're playing on Portuguese (Brazil).
  • startDialogue('dialogue', 'breakfast') - Same as above, but plays "Breakfast" as background music.

startVideo

startVideo(videoFile:String, ?canSkip:Bool = true, ?forMidSong:Bool = false, ?shouldLoop:Bool = false, ?playOnLoad:Bool = true):Void

Plays a Video Cutscene.

  • videoFile - Video .MP4 file located inside "videos/" folder, don't include the extension!
  • canSkip (Optional) - Allows the video to be skipped by holding down Enter for a second. Defaults to true.
  • forMidSong (Optional) - Doesn't start/finish song after the video ends, needed for mid-song cutscenes. Defaults to false.
  • shouldLoop (Optional) - Video will repeat once finished. Defaults to false.
  • playOnLoad (Optional) - Videos plays instantly after loading. Defaults to true.

Examples:

  • startVideo('test_video') - Plays the video mods/My-Mod/videos/test_video.mp4.

makeLuaCharacter

makeLuaCharacter(tag:String, ?json:String = 'bf', ?x:Float = 0, ?y:Float = 0, ?chart:String = '', ?skipDance:Bool = false, ?danceEveryNumBeats:Int):Void

Makes a new character, tied to a chart with an option to change the amount of beats before dancing. (NOT FULLY IMPLEMENTED)

  • tag - Sprite name.
  • json (Optional) - name of the character to load.
  • x (Optional) - X position.
  • y (Optional) - Y position.
  • chart (Optional) - Name of the chart file to load in data/songName/
  • skipDance (Optional) - Whether the character idles or not.
  • danceEveryNumBeats (Optional) - the number of beats before the character idles.

Examples:

  • makeLuaCharacter('picoSpeak', 'pico-speaker', 0, 0, 'picoSpeaker', true, 0) Will make a character using the 'pico-speaker' json following the chart 'picoSpeaker'.
